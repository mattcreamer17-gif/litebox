# Litebox landing page

A bespoke, one-page partnership read prepared for Guillermo Gonzalez Aleman and the Litebox team. Visual language, color palette, typography, and voice are pulled from Litebox's own site via the brand pack (cream background, deep green display type, geometric accents, Rebond Grotesque / Neue Montreal families with safe fallbacks). No external fonts are loaded. No JavaScript.

## Intended URL path

`/for/litebox`

## Files

- `index.html`. Single-page site, brand-matched chrome, hero through CTA.
- `styles.css`. All tokens sourced from `../Brand Pack/brand-pack.json`.
- `logo.svg`. Copied from `../Brand Pack/assets/logo.svg`.
